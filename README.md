# Sistema de Gestión de Autorizaciones Municipales - NGR

## 📋 Descripción del Proyecto
Plataforma web interna desarrollada para el área de Desarrollo Inmobiliario de NGR. 
Su objetivo principal es centralizar y controlar el ciclo de vida de los certificados 
ITSE y Licencias de Funcionamiento de la cartera de locales (Bembos, Dunkin', etc.). 
El sistema automatiza el cálculo de SLA y emite alertas preventivas de vencimiento 
mediante un Dashboard central, mitigando activamente los riesgos de clausura.

## 🚀 Stack Tecnológico
* **Frontend:** HTML5, CSS3, JavaScript (Bootstrap 5)
* **Backend:** PHP puro
* **Base de Datos:** MySQL
* **Entorno de Desarrollo:** XAMPP, Visual Studio Code


## 👥 Roles y Permisos Implementados
* **Administrador:** Gestión total de usuarios y accesos del sistema.
* **Jefatura (Malls/Puerta Calle):** Visualización del Dashboard estratégico y métricas.
* **Analista AM:** Registro de fechas, locales, cálculos de SLA y cambio de estados.
